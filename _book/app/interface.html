用户注册接口

接口地址：http://119.23.248.130:8080/api/app/user/register

请求参数

{
    "mobile":"13777770000",
   "password":"a123456",
    "verifyCode":"666666"//手机验证码(前端验证先不不用填写)
}

响应参数

{
    "code":"0",//0表示成功
    "msg":""
}

        ]

}
