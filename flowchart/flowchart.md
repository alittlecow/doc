#### 个人账户充值流程

```
APP充值请求-->生成待支付订单 -->返回待支付订单ID

```

####个人账户支付流程
```
APP请求支付订单-->检查订单状态-->检查账户余额-->账户扣款-->记录用户账户历史-->更改订单状态-->记录总账户流水
```
